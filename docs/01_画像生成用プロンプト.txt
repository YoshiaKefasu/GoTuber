■ このドキュメントについて

GoTuber (および前身 tomari-guruguru) のデフォルトキャラクターを、画像生成 AI で
作るためのプロンプト集です。GoTuber では `docs/新キャラ差し替え手順.md` の Step 1
(6 枚の 4500×4500 親画像を描く) を、画像生成 AI で自動化する手順となっています。

オリジナル (tomari-guruguru) では「ChatGPT Images 2.0」を使う手順でしたが、
Stable Diffusion、NovelAI、Midjourney など、他の画像生成 AI でも同じプロンプトが
使えます。各 AI サービスで "n=3" の指定方法 (生成枚数) や添付機能の利用方法を
読み替えてください。


================================================================================
■ 元プロンプト (tomari-guruguru より MIT 継承)
================================================================================

ChatGPT Images 2.0 を使って画像を生成します（画像生成時の思考モードは「最高」に設定）。
リポジトリ内の「01_画像生成用テンプレ.png」と「作りたいキャラクター画像」を添付して開始してください。
画像は1回に付き2～3枚ほど生成して良いものを選んでね（n=2とかn=3の部分）。


■最初の指示
----------

添付されたキャラクター参照画像と、5×5顔角度テンプレート画像を参照して、同じキャラクターの5×5顔角度リファレンスシートを作成してください。

キャラクター参照画像から、髪型、髪の長さ、前髪、髪飾り、目の形、目の色、服装、襟、リボン、小物、配色、線画、塗り、絵柄をできる限り維持してください。
5×5顔角度テンプレート画像から、各マスの顔の向き、角度、配置、サイズ感、余白、グリッド構成を参照してください。

画像全体は正方形。25個の頭部アイコンを5列×5行で均等に配置してください。背景はシンプルなグレー。
すべて同じキャラクター、同じ絵柄、同じ線画、同じ髪型、同じ髪飾り、同じ服装、同じ色で統一してください。
各マスではキャラクターの顔の角度だけを変えてください。

5×5の角度構成は以下の通りにしてください。

列方向（左から右）：
1列目：左向き横顔
2列目：左斜め45度
3列目：正面
4列目：右斜め45度
5列目：右向き横顔

行方向（上から下）：
1行目：強く上を向く
2行目：少し上を向く
3行目：水平・正面高さ
4行目：少し下を向く
5行目：強く下を向く

つまり各マスは、列の左右角度と行の上下角度を組み合わせた顔向きにしてください。
例：左上は「左向き横顔＋強く上向き」、中央は「正面＋水平」、右下は「右向き横顔＋強く下向き」です。

重要：
全ての顔の大きさ、頭部のスケール、配置バランスをできる限り統一してください。
どのマスの顔も同じくらいの大きさに見えるようにしてください。
顔が大きすぎたり小さすぎたりしないように、全体のバウンディングボックスを揃えてください。

表情指定：
全てのマスで目は開けてください。
全てのマスで口は閉じてください。
口を開けた表情、笑いすぎた表情、泣き顔、怒り顔、ウインク、目閉じ、半目は避けてください。
表情は自然でかわいいニュートラル表情に統一してください。

禁止事項：
別キャラクターにしない。
髪型を変えない。
髪飾りを変えない。
服装を変えない。
目の色を変えない。
表情を大きく変えない。
体全体を描かない。
背景を複雑にしない。
文字、番号、ロゴ、透かしを入れない。
5×5以外の配置にしない。
同じ角度を繰り返さない。
顔のサイズをバラバラにしない。

chibi anime character head angle reference sheet, 5x5 grid, same character, same design, same hairstyle, same accessories, same outfit, same colors, eyes open, closed mouth, neutral cute expression, yaw and pitch matrix, consistent head scale, consistent face size, evenly spaced, gray background, clean line art, soft cel shading, no text, no logo, no extra objects.

ちびキャラ風にして　n=3

----------

■2回目以降で使える指示（目と口の差分を作る）
----------

この画像の全てのポジションの目を閉じている目にして欲しい。目以外は変更しないで
n=2

----------

この画像の全てのポジションの開けている口を「あ」の口で開けている口にして欲しい。口以外は変更しないで
n=2

----------

この画像の全てのポジションの開けている口小さく開けている口にして欲しい。口以外は変更しないで
n=2

----------

■上記のような指示で画像を生成し、良い出来のものを選んで6枚の表情差分を作成します。その後、4倍にアップスケール。背景透過処理をして、以下のファイル名で保存してください。
アップスケーラーは、私は以下を使わせていただいています。
https://github.com/tsukumijima/Real-ESRGAN-GUI

A_目開け_口とじ.png
B_目開け_口中間.png
C_目開け_口開け.png
D_目閉じ_口とじ.png
E_目閉じ_口中間.png
F_目閉じ_口開け.png

※品質にこだわりたい場合は、PhotoshopやGIMPなどでベース画像を決めて、目と口の部分をトリミングして組み合わせるのがオススメです。
AI画像生成時に発生する色ムラが目立たなくなり綺麗な仕上がりになります。


================================================================================
■ GoTuber 固有の補足
================================================================================

[1] 5×5 顔角度テンプレ PNG について

    元 tomari-guruguru にも「01_画像生成用テンプレ.png」は**存在しない**ようです
    (リポジトリに含まれていない、または別の場所にある可能性)。
    代わりに、以下のいずれかを「5×5 顔角度テンプレート画像」として添付してください:

    - **自作**: 4500×4500 px のキャンバスに 5×5 = 25 マスのグリッド線を引いた画像
      (マスのサイズは 900×900 px)。各マスは空白で OK、AI が顔を埋めてくれる。
    - **流用**: 既存の 5×5 顔角度リファレンス画像 (ネットで「5×5 face angle
      reference」で検索すると多数見つかる)。
    - **流用 (tomari-guruguru)**: 元 `public/slices2/A/r0c0.webp` などの WebP を
      並べたものをテンプレートとして使える (顔の構図参考用)。

[2] 生成画像のサイズについて

    元プロンプトは「5×5 顔角度リファレンスシート」1 枚を生成する手順のため、
    **出力サイズが 4500×4500 px ちょうどになる保証はありません**。
    生成後に Photoshop / GIMP / Krita / Pillow 等で 4500×4500 px にクロップ・
    リサイズしてからスライスツールに渡してください。

[3] GoTuber のスライスツール

    6 枚の 4500×4500 親画像 (A〜F) を生成したら、GoTuber 同梱の
    `tools/slice_character_sheets.py` で 1200×1200 anchored WebP に切り出します。
    詳細は `docs/新キャラ差し替え手順.md` を参照。

    簡単な実行例:

        python tools/slice_character_sheets.py \
          --source "新キャラ資料" \
          --slices-out "assets/characters/mychara" \
          --format webp \
          --jobs 4 \
          --component-mode \
          --min-component-area 80 \
          --alpha-threshold 64 \
          --remove-gray-residue

[4] ライセンスの注意

    AI 生成画像の商用利用可否は各 AI サービスの規約に依存します。Stable Diffusion
    系のローカル実行は比較的自由、ChatGPT Images / Midjourney / NovelAI は
    商用利用に制限がある場合があります。GoTuber 自体は MIT ライセンスですが、
    生成したキャラクター画像のライセンスは AI サービス規約に従ってください。

[5] 推奨ワークフロー (要約)

    1. AI で 5×5 顔角度リファレンスシート (A: 目開け・口とじ) を生成
    2. 目を閉じた版 (D) と口を開けた版 (C, B) を追加生成
    3. 計 6 枚 (A, B, C, D, E, F) を選ぶ
    4. 4 倍アップスケール + 背景透過
    5. ファイル名規約で保存
    6. GoTuber の `tools/slice_character_sheets.py` でスライス
    7. `config/default.yaml` の `basePath` を新キャラのパスに変更
    8. `.\bin\gotuber.exe` で起動して確認
